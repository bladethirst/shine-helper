## 3. 功能设计

### 3.1 用户交互

1. **点击麦克风按钮** - 开始/停止录音
2. **快捷键触发** - Ctrl+V 开始录音，再次按停止
3. **实时转录** - 语音识别结果实时显示在输入框
4. **静音检测** - 3秒无声音自动停止录音

### 3.2 状态指示

| 状态 | 描述 | UI 表现 |
|------|------|----------|
| idle | 空闲 | 麦克风图标 normal |
| listening | 聆听中 | 麦克风图标 pulse 动画 |
| processing | 处理中 | 麦克风图标 loading |
| error | 错误 | 麦克风图标红色，显示错误信息 |

### 3.3 配置项

在配置页面添加 Vosk 服务配置：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| vosk.url | WebSocket 服务地址 | ws://localhost:5000 |
| vosk.apiKey | API Key | (空) |
| vosk.enabled | 是否启用语音输入 | false |
| vosk.silenceTimeout | 静音超时时间(ms) | 3000 |
